# Work Polish

把大白话变成听起来很专业的职场语言。

## 使用方式

```
/work-polish --轻度 帮老王把登录页面的颜色改了，顺便加了个 loading 动画
/work-polish --重度 明天要上线但还有 3 个 bug 没修完
```

## 模式说明

### 轻度包装

适度提升专业感，见好就收，适合日常周报、站立会、聊天。

**原则：**
- 加 2-3 个行业术语或动词，不要过度
- 保持原意，不改变事实
- 听起来像一个正常说话的成年人
- 适当的结构化（分点、加粗关键词）

**常用词替换参考（按需使用，不要全塞进去）：**
- 帮 XX 改了 → 协助 XX 完成了 XX 的优化
- 修了个 bug → 修复了 XX 场景下的问题
- 加了个功能 → 完成了 XX 功能的开发
- 改了下 UI → 对 XX 界面进行了体验优化
- 问了下 XX → 与 XX 进行了沟通 / 对齐
- 差不多了 → 已基本完成
- 还没弄完 → 正在推进中
- 发现一个问题 → 在验证过程中发现 XX 存在风险点

### 重度包装

全力输出，适合 PPT 标题、季度汇报、向上汇报。要让老板觉得你做的事情很值钱。

**原则：**
- 动词必须是"赋能、聚焦、驱动、沉淀、拉通、闭环、抓手、深耕"这类
- 名词要带"生态、链路、颗粒度、水位、体感、心智、护城河"等
- 任何事情都要提炼成"方法论"或"体系"
- 数字一定要有（实在没有就写"显著"）
- 每句话都像 PPT 标题
- 不要怕夸张，老板就吃这一套

**重度包装输出格式：**
- 输出 1-2 个版本供选择
- 标注：短标题（PPT用）+ 长描述（汇报口述用）

**经典句式：**
- 通过 XX，实现了 XX 的 XX 化 / 体系化
- 聚焦 XX 核心场景，打造 XX 闭环
- 以 XX 为抓手，驱动 XX 增长
- 深耕 XX 领域，沉淀 XX 方法论
- 完成了 XX 的从 0 到 1 搭建

## 输出要求

1. 先简要说明改了什么、为什么这样改（1-2 句）
2. 然后输出润色后的结果，用代码块或引用块高亮
3. 轻度模式直接输出一段话
4. 重度模式输出：PPT 短标题 + 口述长版本

## 触发词

工作汇报、润色、写周报、汇报、职场包装、PPT 标题
